# Melanoma genomic meta-analysis: code and study-level data

This repository contains the original R analysis script and the study-level dataset shared during peer review for the manuscript on genomic alterations in cutaneous melanoma.

## Files in this repository

- `meta mutation.R` — original R script used in the analytical workflow.
- `Table S11 Data from Studies Included in the Meta-analysis.xlsx` — study-level dataset used as the primary source dataset for the meta-analysis.
- `Table S12 List of Studies and Reasons for Exclusion.xlsx` — excluded studies and reasons for exclusion.

## Important note

The repository is shared to improve transparency and reproducibility during peer review. The R script reflects the original analytical workflow. Some filenames referenced inside the historical script correspond to intermediate working files created during the original analysis process. For study-level metadata in the revised submission, the authoritative source dataset is `Table S11 Data from Studies Included in the Meta-analysis.xlsx`.

## How to cite in the manuscript

Code Availability: The original R analysis script and supporting study-level data used in this study are publicly available in this GitHub repository.
